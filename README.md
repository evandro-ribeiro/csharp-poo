# Fundamentos de Programação Orientada a Objetos em C#

## Contexto

Modelagem, utilizando POO, de um sistema que trabalha com celulares. Para isso, foi solicitado que faça uma abstração de um celular e disponibilize maneiras de diferentes marcas e modelos terem seu próprio comportamento, possibilitando um maior reuso de código e usando a orientação a objetos.

## Proposta

Criar um sistema em .NET, do tipo console, mapeando uma classe abstrata e classes específicas para dois tipos de celulares: Nokia e iPhone.
